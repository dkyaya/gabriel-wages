# Library Source Gap List

**Date:** 2026-07-05
**Scope:** desk-research planning only. Lists the highest-value evidence gaps surfaced by `non_safety_library_wage_mechanism_and_corpus_scan_2026-07-05.md`. Several items below already have a partial source identified during this session's bounded web search or corpus read; those are noted. This is a punch list for future sessions, not a completed data build.

For each gap: why it matters, likely source families, a feasibility tag (`desk_research_feasible`, `data_build_feasible`, `likely_hard`), and which dimension it speaks to (wage levels, wage growth, staffing supply, classification, professional credentials, part-time/volunteer buffering, service deferral, budget constraints, governance, or public salience).

## 1. Library vacancy/turnover evidence

**Why it matters:** the most direct staffing-strain measure, and the library-side analog to the police/teacher/DPW/clerical-admin vacancy data already used elsewhere in the project's mechanism map. None of this project's three library CBAs contains vacancy, applicant, or "hard to fill" language.

**Likely source families:** national ALA/PLA Public Library Staff Survey data (already identified this session at the national level: 18% of public libraries reported staff losses in the prior 12 months per the 2024 survey); Massachusetts-specific municipal library annual reports or budget books; MBLC data holdings, if any track staffing at the municipal level (not reviewed this session).

**Feasibility:** `desk_research_feasible` for the national snapshot (already closed this session); `data_build_feasible` for this project's specific cities if their annual town reports or library annual reports include staffing/vacancy detail; `likely_hard` for a systematic multi-city Massachusetts series, since no centralized Massachusetts library-vacancy database was identified this session.

**Dimension:** staffing supply.

## 2. Librarian vs. assistant/page composition, Massachusetts-specific

**Why it matters:** this project's own corpus shows a wide composition spread within library bargaining units (Wayland's seven-grade schedule; Franklin's explicit page exclusion), but no data exists on the actual headcount or FTE split between professional librarians, library assistants, and pages for any of this project's three cities.

**Likely source families:** MBLC's Annual Report Information Survey (ARIS) system, which Massachusetts libraries submit annually as part of state-aid certification (referenced but not directly reviewed this session); individual library annual reports; municipal personnel/HR records.

**Feasibility:** `desk_research_feasible` for confirming whether MBLC's ARIS data includes a staffing-composition breakdown at all (not verified this session); `data_build_feasible` for this project's specific three cities if ARIS or annual-report data is available at that level of detail.

**Dimension:** staffing supply / composition.

## 3. Professional credential requirements, Massachusetts-specific

**Why it matters:** this session found Franklin's CBA pays an explicit MLS/MLIS stipend, but did not verify whether Massachusetts's own librarian-certification regulation (605 CMR 3.00, administered by MBLC) requires an MLS/MLIS for any specific position tier in any of this project's three cities, or only confers a state-aid-eligibility benefit on the municipality rather than a legal requirement for any specific job title.

**Likely source families:** MBLC's own Librarian Certification pages (605 CMR 3.00, identified this session but not read in depth); individual library job postings or personnel bylaws for this project's specific cities (not reviewed).

**Feasibility:** `desk_research_feasible` for a closer reading of 605 CMR 3.00's actual requirements (identified this session, not yet read in depth); `data_build_feasible` for confirming which of this project's three cities' library director or department-head positions require MBLC certification specifically, which was not established this session.

**Dimension:** professional credentials.

## 4. Part-time staffing shares, Massachusetts-specific

**Why it matters:** this project's corpus confirms part-time employment is a core, contractually defined category in all three library units, but contains no data on what *share* of each unit's total headcount or hours is part-time, or whether that share has changed over time.

**Likely source families:** MBLC ARIS data (if it captures staffing composition, not verified); individual library annual reports; municipal budget/personnel records.

**Feasibility:** `likely_hard` at a systematic level, since no centralized source was identified this session; `data_build_feasible` only if narrowed to this project's specific three cities and their own personnel records, which were not reviewed this session beyond the CBA text itself.

**Dimension:** staffing supply / part-time buffering.

## 5. Volunteer use, Massachusetts-specific — PARTIALLY CLOSED 2026-07-05

**Why it matters:** directly tests the `library_part_time_volunteer_buffering` hypothesis's volunteer-substitution sub-claim.

**Status:** partially closed this session. Franklin's Article 28 explicitly prohibits replacing bargaining-unit hours with volunteer help without union agreement — direct, corpus-confirmed evidence *against* the volunteer-substitution mechanism for at least this one city and bargaining unit. Wayland's and Seekonk's documents contain no comparable clause (Wayland's single "volunteer" mention describes staff self-scheduling for optional shifts, not external volunteer substitution; Seekonk's document has no volunteer language at all).

**Remaining sub-gap:** whether any of these three libraries (or others in Massachusetts) actually uses volunteer labor in practice, in what capacity, and whether that use has grown over time — this project's corpus can only show what the *contracts* say about the topic, not actual practice. This remains `likely_hard`, since volunteer-usage data is not typically published in a centralized or comparable format across Massachusetts municipal libraries.

**Dimension:** part-time/volunteer buffering.

## 6. Reclassification/classification studies — PARTIALLY CLOSED 2026-07-05

**Why it matters:** directly tests the `library_classification_reclassification` hypothesis.

**Status:** the process itself is now documented for two of three cities in `non_safety_library_wage_mechanism_and_corpus_scan_2026-07-05.md` §5 — Franklin's Section 14.9 and Wayland's Article 27 both describe a director-initiated review, administrative-authority (Town Administrator or HR Manager) approval, and a new-anniversary-date/step-eligibility consequence. Seekonk's document has no comparable reclassification language.

**Remaining sub-gap:** actual reclassification-request outcome data (approval rates, dollar impact, frequency) remains `likely_hard`, since this is internal HR administrative data, not published in the CBA text itself — the identical remaining sub-gap already flagged for DPW's and clerical/admin's reclassification mechanisms.

**Dimension:** classification / reclassification.

## 7. Pay compression / internal equity, Massachusetts-specific

**Why it matters:** no document in this project's library corpus contains explicit pay-compression or internal-equity language; the only compression-adjacent evidence found this session is a 20-year-old national study (2006 ALA-APA non-MLS salary survey), which cannot be assumed to describe current Massachusetts conditions.

**Likely source families:** current ALA-APA salary survey data, if a more recent edition exists (not checked this session beyond the 2006 study and the general 2024 PLA staff survey, which did not include a compression-specific breakdown); Massachusetts municipal library salary schedules over multiple contract cycles (this project's corpus currently has only one cycle per city, insufficient for a within-city trend).

**Feasibility:** `desk_research_feasible` for checking whether a more recent ALA-APA or PLA compensation-compression study exists (not done this session); `likely_hard` for a Massachusetts-specific or this-project's-cities-specific compression trend, given the single-cycle-per-city corpus limitation.

**Dimension:** pay compression / wage levels.

## 8. Service-hour reductions, branch closures, program cuts — Massachusetts-specific

**Why it matters:** directly tests the `library_service_deferral` hypothesis. No document in this project's corpus contains hours-reduction, branch-closure, or program/acquisitions-cut language — this is expected, since a labor CBA is not the document type that would typically contain this kind of service-level narrative, per the source-document-production-effects finding already established twice for DPW and clerical/admin and now replicated a third time here.

**Likely source families:** municipal library annual reports (if published); MBLC ARIS compliance data (which tracks minimum-standards compliance, potentially including hours-of-service data); local news coverage of specific hours/branch/program changes (illustrative, not systematic proof).

**Feasibility:** `likely_hard` for a systematic multi-city series; `desk_research_feasible` for illustrative individual examples if a specific Massachusetts library is targeted; `data_build_feasible` for this project's three specific cities if their MBLC ARIS filings or annual reports are reviewed directly (not done this session).

**Dimension:** service deferral.

## 9. Municipal/library budget constraints — PARTIALLY CLOSED 2026-07-05

**Why it matters:** tests whether library budgets face a library-specific floor or ceiling mechanism, mirroring the teacher Chapter 70/net-school-spending finding.

**Status:** closed at the general-mechanism level this session. MBLC's Municipal Appropriation Requirement (MAR) is confirmed via MBLC's own certification/regulations pages: a municipality's library appropriation generally may not fall below roughly 102.5% of the average of the preceding three years' appropriations, subject to MBLC waivers, to remain certified for the State Aid to Public Libraries Program.

**Remaining sub-gap:** whether any of this project's three library cities have ever been out of MAR compliance, received a waiver, or had this mechanism bind in a way relevant to library staff wages specifically — not reviewed this session. Would require MBLC compliance-history records or these cities' own budget history.

**Dimension:** budget constraints.

## 10. Trustee or library-board governance effects

**Why it matters:** Seekonk's CBA names the Board of Library Trustees in its management-rights clause; MBLC requires the Trustees chair's signature on the annual state-aid compliance form. Neither of these facts establishes whether trustees have any independent budget or wage-setting authority distinct from the ordinary town administration.

**Likely source families:** municipal charters or library-specific enabling bylaws/statutes (M.G.L. c. 78, the general public-library-trustees statute, not reviewed this session); individual town charters for this project's three cities (not reviewed).

**Feasibility:** `desk_research_feasible` for a general reading of M.G.L. c. 78's trustee-authority provisions (not done this session, flagged as a bounded follow-up if pursued); `data_build_feasible` for confirming any of this project's three cities' specific trustee-authority arrangements, which was not established this session.

**Dimension:** governance.

## 11. Public support vs. wage translation, Massachusetts-specific

**Why it matters:** the national context (§9 of the mechanism memo) shows library compensation stagnation alongside generally positive public regard for libraries nationally, but no Massachusetts-specific or this-project's-cities-specific public-opinion or wage-trend data was reviewed to test whether the same pattern holds locally.

**Likely source families:** general public-opinion survey data on libraries (national, not Massachusetts-specific, and not pulled this session beyond general awareness); this project's own future wage-trend work, if a multi-cycle library corpus is ever assembled.

**Feasibility:** `likely_hard` for a genuinely Massachusetts-specific or this-project's-cities-specific test; `desk_research_feasible` only for further national-level public-opinion context, which would not resolve the local question.

**Dimension:** public salience / wage translation.

## 14. Massachusetts library bargaining impasse-process verification — CLOSED 2026-07-05

**Why it matters:** tests whether Massachusetts library workers, like teachers, DPW, and clerical/admin, follow the ordinary Chapter 150E Section 9 route with no JLMC-equivalent backstop.

**Status:** closed in `non_safety_library_wage_mechanism_and_corpus_scan_2026-07-05.md` §8. The Massachusetts Municipal Association's own Select Board Handbook states directly that "it is common for clerical and library employees to be union members" under the ordinary Chapter 150E framework, explicitly grouping library employees with clerical staff in the non-JLMC category. This project's own corpus independently corroborates the finding: all three `library` rows' `binding_arbitration_statute` fields cite MA G.L. c. 150E, and every arbitration clause found is grievance-scoped. This verification was folded into the combined mechanism-and-corpus-scan memo rather than given its own dedicated institutional memo, since it required comparatively little new research given the MMA Handbook's direct, explicit language.

**Remaining sub-gap:** a closer reading of M.G.L. c. 78 (the general public-library-trustees statute) was not done this session and would be needed for a more complete picture of trustee governance authority specifically (see gap 10 above), though this does not bear on the impasse-process finding itself.
