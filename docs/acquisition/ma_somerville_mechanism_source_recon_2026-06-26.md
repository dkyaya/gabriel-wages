## Evidence Classification

The strongest Somerville non-safety item found in this pass is the 2025 SEU ratification press release. It gives public wage rationale for paraprofessional pay and frames the settlement in regional labor-campaign terms, but it is not a formal peer-wage comparison exhibit and is not a final agreement.

The 2025-2028 SEU CBAs are clean public causal candidates if the project later wants educator CBA scale. They are not the missing award/factfinding-style reasoning source. Generic CBA language about comparable work, comparable training, or comparable equipment should not be read as peer-wage comparability.

## Corpus-Handling Recommendation

Do not ingest Somerville materials in this pass. The ratification release belongs in a future discourse/proxy system if one is added. The CBAs should remain causal-corpus leads, not current mechanism-source additions, because this task is not a broad CBA expansion pass.
