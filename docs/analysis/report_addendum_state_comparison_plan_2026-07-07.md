# Report Addendum Planning Memo — Texas/Ohio State Comparison

**Date:** 2026-07-07
**Type:** planning memo only. Explains how the Texas/Ohio institutional scan (`texas_ohio_state_comparison_institutional_scan_2026-07-07.md`) should feed the existing report draft (`report_draft_safety_non_safety_wage_mechanisms_2026-07-06.md`). Proposes no edits to the report draft itself this session — that remains a separate, future editorial step, consistent with this run's scoping-only boundary.

## 1. Whether to add a short "Why look beyond Massachusetts?" section

**Recommendation: yes, but keep it short (2-4 sentences) and place it as a new subsection within the existing "Massachusetts and National Nuance" section, not as a new top-level section.** The report already has a section built for exactly this kind of institutional-generalizability discussion (it currently covers the three Massachusetts arbitration regimes — JLMC, ordinary Chapter 150E, and the MBTA's c. 161A — plus the Iowa counterexample from the national citation audit). A short addition naming the PI's own motivating concern (that JLMC-specific critiques of Massachusetts wage growth should not be mistaken for a general public-safety finding) fits naturally there, rather than requiring a new report section with its own claim hierarchy and evidence map entry.

## 2. Whether Texas/Ohio should be included in main text or appendix

**Recommendation: main text, but only as a short, clearly-scoped paragraph — not a table, not a group-by-group treatment, and not yet an Evidence Map row.** The Evidence Map and Group-by-Group Findings sections are structured around this project's own corpus (32 rows, 9 Massachusetts cities); Texas and Ohio have zero corpus rows and should not be given a same-format entry that would visually imply comparable evidentiary weight. A short main-text paragraph, clearly labeled as "current scoping status, not yet corpus-backed," keeps the finding visible to a reader without overstating it. The full institutional detail (statute citations, city-by-city adoption variation, the Chapter 174/142/146 distinctions, Ohio's no-strike list) belongs in an appendix pointer to this session's own memo, not reproduced in the main text.

## 3. Proposed paragraph or subsection language for the report draft

The following is proposed language only — not yet inserted into the report draft. If approved, it should be added as a new final subsection within "Massachusetts and National Nuance," after the existing Iowa-counterexample discussion:

> **Why look beyond Massachusetts?** Every finding in this report to this point rests on Massachusetts's JLMC-versus-Chapter-150E institutional split, and a reasonable reader might ask whether the safety/non-safety wage-mechanism gap this project documents is a general public-safety phenomenon or an artifact of Massachusetts's specific compulsory-arbitration statute. A bounded scoping pass (not yet corpus-backed — see `texas_ohio_state_comparison_institutional_scan_2026-07-07.md`) examined Texas and Ohio as two additional, deliberately different contrast cases: Texas's default rule prohibits public-sector collective bargaining entirely, with police/fire receiving a narrow, city-by-city optional bargaining channel that is itself not compulsorily arbitrated; Ohio embeds a safety/non-safety split within a single statewide statute, trading a strike right for non-safety employees against compulsory conciliation for statutorily named safety forces. Neither state has yet contributed a single collected document to this project's corpus, and this paragraph should not be read as extending this report's findings beyond Massachusetts — it identifies two promising, differently-structured next comparison cases, not a completed comparison.

This language is deliberately hedged (three separate "not yet corpus-backed" signals) and should be reviewed by the PI before insertion, per the checklist item below.

## 5. How to phrase the PI request without overclaiming

Recommended framing for any report or memo language describing this session's origin: **"At the PI's request, this project conducted a bounded institutional-law scoping pass on Texas and Ohio, to test whether Massachusetts's safety/non-safety wage-mechanism findings depend on Massachusetts's specific JLMC arbitration statute."** Avoid framing that implies:

- that Texas and Ohio were chosen as statistically representative states (they were suggested by the PI as illustrative contrasts, and are described that way in this session's own memo);
- that any wage, staffing, or compensation *finding* (as opposed to an institutional-law finding) exists yet for either state;
- that this scoping pass constitutes or approximates a 50-state survey (it does not — it is two additional states on top of the project's existing 14-state citation audit, which itself was explicitly bounded and non-exhaustive).
