# Non-Safety Wage Mechanism and Corpus Scan: Library Workers

**Date:** 2026-07-05
**Scope:** context-building, institutional-verification, and internal-corpus-review only. No GABRIEL run, no model/API calls, no OEWS/BLS data build, no ingestion, no causal claim. Focuses on Massachusetts municipal library workers where public sources and already-collected corpus files are available, with bounded national context for general mechanisms. Does not attempt comprehensive statewide coverage.

## 1. Purpose and scope

This memo exists to develop library workers as the fourth non-safety comparison group in this project's sequenced roadmap, following teachers, DPW/public works, and clerical/admin, per the recommendation in `wage_mechanism_project_checkpoint_2026-07-05.md` §9. Unlike the first three non-safety groups, library workers can be developed at full depth — mechanism refinement, institutional verification, and a complete corpus scan — in a single session, because all three of this project's already-collected `library` occupation-class rows require no new source acquisition.

This memo focuses on Massachusetts municipal library workers, using bounded national context (American Library Association / Public Library Association survey data, BLS occupational data, a 2006 ALA-APA non-MLS salary study) only to identify general mechanisms that are not Massachusetts-specific. It reviews the three existing `library` corpus rows and their source documents directly but does not ingest any new document, add any new row, or run GABRIEL. It is mechanism discovery and institutional fact-finding only — **it does not make a causal claim** about why library wages behave the way they do, or about how library wages compare causally to police, fire, teacher, DPW, or clerical/admin wages.

## 2. Why library workers matter for the project

Library workers are the natural fourth non-safety group for two distinct reasons, one substantive and one practical.

**Substantively**, library workers test a combination of mechanisms none of the first three non-safety groups tests cleanly on its own:
- **Lower public salience than police/fire and, plausibly, than teachers** — a library service disruption (reduced hours, a closed branch, a longer hold queue) is diffuse and rarely a single dramatic incident, closer to DPW's and clerical/admin's low-salience profile than to teachers' more visible classroom disruptions.
- **Professionalization and credentialing without necessarily strong wage premiums** — the Master of Library Science/Library and Information Science (MLS/MLIS) degree is a real professional credential, structurally analogous to teacher licensure, but (per the national context in §9) the wage premium it confers over non-credentialed library staff has historically been thin, and a genuine minority of public library staff hold it. This tests whether *credentialing alone*, absent a compulsory-arbitration backstop or strong political salience, actually produces the wage floor teachers' credentialing has been hypothesized to support.
- **Formal classification/reclassification systems**, directly comparable to DPW's and clerical/admin's, but embedded in a smaller, more specialized department.
- **Part-time and non-standard staffing as a structural feature of the occupation itself**, not merely an occasional buffering response — unlike teachers (mostly full-time, with substitutes as a distinct, separate labor pool) or DPW (mostly full-time, with contractors as an external substitute), library bargaining units in this project's own corpus include regular part-time employees as core, permanent members of the same bargaining unit, working alongside full-time colleagues under the same classification schedule (confirmed corpus-wide, §5 below).
- **Library-specific budget/governance structures** — Massachusetts municipal libraries participate in a distinct state certification and municipal-appropriation regime (the Massachusetts Board of Library Commissioners' Municipal Appropriation Requirement) that has no direct analog in DPW's or clerical/admin's ordinary general-fund budgeting and is structurally different from (weaker than) teachers' Chapter 70/net-school-spending floor (§8).

**Practically**, library is the only remaining non-safety occupation class in this project's schema with already-collected corpus rows: three `library` rows (Seekonk, Franklin, Wayland) already exist in `data/contracts.csv`, all three already coverage-matched to safety-unit comparison pairs in `data/city_coverage.csv`. No other non-safety group (sanitation, transit, parks/rec, nurse_health) has any rows collected yet. This makes library the only group where a full mechanism-refinement-plus-corpus-scan session can run in one pass, with no source-acquisition step in between, mirroring the two-stage pattern (refinement memo, then corpus-scan memo) already applied to DPW and clerical/admin — collapsed here into one memo since both stages draw on the same three documents.

**What library workers should not be collapsed into:** library staff share some surface features with clerical/admin (classification systems, low physical risk, low emergency-response profile) and some with teachers (public-facing, credentialed at the professional tier, a civic/educational mission), but this memo treats library as its own occupation class throughout, consistent with the project's existing `occupation_class` controlled vocabulary (`library` is already a distinct value, separate from `clerical_admin`). Collapsing library into clerical/admin would erase the two mechanisms most specific to library work — professional MLS/MLIS credentialing and structural part-time/page staffing — that this memo exists to examine.

## 3. Library occupation/classification map

Library work spans a range of titles, credential levels, and staffing statuses that this project's own corpus already illustrates directly (see §5):

- **Librarian / professional librarian** — the credentialed tier, typically requiring or preferring an MLS/MLIS degree; in this project's corpus, titles at this tier include Seekonk's "Senior Librarian" and "Staff Librarian I/II" (renamed "Librarian I" in the current contract) and Wayland's "Bibliographic Services Librarian" and "Reference Librarian."
- **Children's librarian** — a specialized public-service role distinct from general reference or circulation work; confirmed in this project's corpus at Wayland ("Children's Librarian," "Assistant Children's Librarian") and Franklin ("Youth Services Librarian," "Assistant Youth Services Librarian").
- **Reference librarian** — confirmed at Wayland ("Reference Librarian," "Computer Specialist/Reference Librarian" — a title that itself signals the digital-services workload shift discussed in §6).
- **Circulation staff** — confirmed at Wayland ("Head of Circulation," "Library Assistant - Circulation"); Seekonk's title list includes a "Customer Service/Inventory Supervisor," a circulation-adjacent supervisory role.
- **Library assistant** — the general paraprofessional tier, typically not requiring a professional degree; confirmed at Wayland ("Library Assistant," the L-1 grade, and "Library Assistant - Children's").
- **Page** — an entry-level, typically part-time, typically youngest-staff role; **confirmed present in this project's corpus only as an explicit exclusion**, not as a covered title (Franklin's recognition clause explicitly excludes "pages" from the bargaining unit — see §5). No page-specific wage or classification data exists in this project's corpus as a result.
- **Technical services / cataloging** — confirmed at Seekonk ("Technical Services Associates II") and at Wayland via "Bibliographic Services Librarian" and "Library Associate - Periodicals."
- **Branch managers / supervisors** — not directly observed as a distinct "branch manager" title in this project's corpus (all three cities appear to be single-building libraries, not branch systems), though supervisory tiers exist within each unit (Seekonk's "Customer Service/Inventory Supervisor"; Wayland's L-7 "Assistant Director").
- **Administrative/business roles nested inside the library** — confirmed at Wayland ("Administrative Assistant," "Business Manager," and, notably, "Custodian" — a facilities role folded into the same library-specific bargaining unit and pay grid, at its own "L-C" grade).
- **Part-time staff** — confirmed as a core, contractually defined category (not merely an exception) in all three corpus documents (§5).

**Why library should not be treated as one homogeneous occupation, stated as a concrete corpus finding rather than an assumption:** Wayland's own ten-step, seven-grade-plus-custodian classification schedule (L-1 through L-7 plus L-C) spans an entry Library Assistant (L-1, $41,305–$53,872 at FY2020 rates before subsequent COLAs) to an Assistant Director (L-7, $65,549–$85,506 at the same rates) — a wider single-unit pay-grade spread than this project has documented for any other non-safety occupation class to date, DPW included. A single "library wage" figure would blend this entire spread, plus a custodian classification with no library-specific duties at all, into one number — a composition risk at least as severe as the DPW classification-fragmentation and teacher grade/subject-composition confounds this project has already flagged.

## 5. Existing library corpus findings

### Classification/pay-grade structure

All three documents have an explicit classification-and-pay-grade structure, but at very different levels of granularity:

- **Wayland** has by far the richest structure found in this scan: a seven-tier grade system (L-1 through L-7) plus a separate custodian grade (L-C), each with a ten-step schedule, explicitly mapped to titles in the contract's own appendix (L-1 = Library Assistant; L-2 = Library Associate; L-4 = Bibliographic Services, Library Administrative, Computer Specialist/Reference, Assistant Children's, Head of Circulation; L-5 = Children's, Business Manager; L-7 = Assistant Director; L-C = Custodian). This is a materially more granular single-unit classification structure than either Seekonk's or Franklin's, and more granular than most of the DPW and clerical/admin classification systems already documented elsewhere in this project's corpus.
- **Franklin** has a three-tier system (L1, L2, L3) with a four-step schedule per tier, tied to a title list that includes "Assistant Youth Services Librarian," "Programming and Community Outreach Librarian," "Youth Services Librarian," and "Library Assistant" (exact tier-to-title mapping not fully resolved from the extracted text in this session; the appendix table structure suggests L1 is the lowest tier given its lower dollar figures).
- **Seekonk** has the least granular structure of the three: a four-step schedule applied to a short title list (Senior Librarian, Customer Service/Inventory Supervisor, Staff Librarian I, Staff Librarian II — since renamed "Librarian I" — Library Administrative Associate, Technical Services Associate II), with no separate numbered "grade" tier visible in the extracted salary-schedule text beyond the step numbers themselves.

### Professional credentials (MLS/MLIS)

**Franklin is the only one of the three documents with an explicit, corpus-confirmed professional-credential-to-pay linkage.** Franklin's Section 15.2 ("Educational Incentive") pays a tiered, permanent per-pay-period stipend to any employee holding a degree from an accredited institution: Associate's Degree ($26.93/pay period, $700/year), Bachelor's Degree ($61.54/pay period, $1,600/year), and — at the same, highest tier — Master of Library Science (MLS) or Master of Library Information Science (MLIS) ($73.08/pay period, $1,900/year). This is a direct, verbatim, corpus-confirmed example of the professional-credentialing-to-pay mechanism this memo set out to test, and it is structured as a fixed add-on stipend layered on top of the base step-and-grade schedule, not a separate, higher base-pay track for MLS holders specifically.

Wayland's contract references a Master's degree only in the context of educational-leave and tuition-reimbursement benefits (paid leave to attend a class "if pursuing a Master's degree in library science," and up to $1,300 reimbursement toward "a graduate level course at an accredited college") — a real, tangible educational-investment benefit, but not a permanent stipend or pay-grade premium tied to already holding the degree, a materially different (and weaker, in ongoing-wage terms) mechanism than Franklin's.

Seekonk's document contains **no** degree, education, MLS, or MLIS language at all in the extracted text — a genuine absence, not a search-method limitation, since the same search terms found rich results in both other documents.

### Part-time staff, pages, and volunteers

All three documents treat **part-time employment as a core, contractually defined bargaining-unit category**, not an edge case:
- Seekonk: "regular part-time employees" are named directly in the recognition clause alongside full-time employees; part-time employees "normally will work a maximum" of some threshold (specific number not fully resolved from this session's extraction) and separately qualify for prorated holiday pay (at 20+ hours/week), prorated sick/family leave, and prorated longevity pay.
- Franklin: part-time employees are guaranteed a contractual minimum of "ten (10) hours per week," and receive prorated stipends, prorated paid time off (with distinct rules for those working 19 or fewer vs. 20 or more hours/week), consistent with a genuinely graduated, multi-tier part-time structure rather than a single "part-time" catch-all.
- Wayland: the recognition clause itself defines unit membership by an hours threshold ("all regular full-time and part-time employees who regularly work ten hours or more each week"), meaning employees working fewer than ten hours/week (the most page-like tier) are **not covered by this CBA at all** — directly relevant to the page/volunteer-buffering discussion in §7.

**Pages are explicitly excluded from bargaining-unit coverage, not merely absent from the documents.** Franklin's Article 1 recognition clause names "pages" directly as an excluded category, alongside the Library Director, Assistant Library Director, Administrative Executive Secretary, and custodial/maintenance personnel. This is the clearest, most direct evidence in this scan that at least one library-specific occupational tier (pages) sits entirely outside this project's `library`-classified corpus rows' own wage data — a page-specific wage figure, where it exists, would need a different source entirely (municipal personnel bylaws, non-union pay plans, or job postings), not this project's collected CBAs.

**Volunteers appear in the text, but not as a labor-substitution mechanism in two of the three documents, and as an explicitly *protected-against* mechanism in the third.** Wayland's single "volunteer" mention describes employees *themselves* volunteering for optional Saturday summer shifts (a scheduling mechanism, not an external volunteer-labor substitute) — this should not be read as evidence of volunteer-labor buffering. Seekonk's document contains no "volunteer" language at all. **Franklin's Article 28 ("Volunteer Work") is the one directly on-point clause found in this scan: "Under no circumstances will bargaining unit members' hours be replaced by volunteer help without agreement of the Union."** This is strong, direct, corpus-confirmed evidence that at least one of this project's library bargaining units has explicitly negotiated *against* the volunteer-substitution mechanism this memo hypothesizes as a possible wage-restraint channel (§7) — the union has anticipated and contractually foreclosed the exact buffering channel a management-rights-maximizing employer might otherwise use, which is itself informative about how seriously that risk is taken by at least one bargaining unit.

### Reclassification/classification review

**Reclassification is a real, corpus-confirmed mechanism in two of the three documents, with near-identical structure.** Franklin's Section 14.9 and Wayland's Article 27 are almost procedurally identical: an employee's duties may be reviewed by the Library Director (on the employee's request or the Director's own initiative); if the Director finds the employee belongs in a different classification, the Director recommends reclassification to a higher administrative authority (Franklin: the Town Administrator; Wayland: the Human Resources Manager) for approval or denial; a reclassified employee receives a new anniversary date and becomes eligible for a step increase on the next anniversary following reclassification. Both documents also address temporary higher-classification work assignments (pay at the higher rate after a minimum duration, generally two weeks) as a distinct, separate mechanism from permanent reclassification. Seekonk's document contains **no** reclassification language in the extracted text — a genuine absence, consistent with its generally shorter, less procedurally detailed structure relative to the other two.

### Pay compression / internal equity

No document in this scan contains explicit "pay compression," "salary inversion," or "internal equity" language. This mirrors the identical absence already documented in both the DPW and clerical/admin corpus scans — a third independent replication of the same finding.

### Overtime / premium pay

All three documents have standard overtime provisions (time-and-a-half after a defined weekly threshold, generally 35 or 40 hours depending on the city), with Franklin explicitly stating overtime work is voluntary and that no employee may be discriminated against for declining it. No document contains minimum-staffing, callback, or emergency-response overtime language of the kind found extensively in this project's DPW and police/fire corpus — overtime in these three documents reads as an ordinary, non-emergency labor-standards provision, not an operational-necessity mechanism.

### Management rights

Seekonk's Article III management-rights clause is broad and standard (the Town, "acting through its Board of Selectmen, Board of Library Trustees, Town Administrator, Department Heads or other appropriate officials," retains rights to add/eliminate departments and increase/decrease jobs, among others) — notable specifically for naming the **Board of Library Trustees** as one of several bodies through which management rights are exercised, alongside (not instead of) the ordinary municipal chain of command. Franklin's and Wayland's management-rights framing is present but less explicitly multi-body in the extracted text.

### Service hours, staffing levels, layoffs

No document contains explicit "branch hours," "service hours," or public-facing hours-of-operation language in the sections reviewed (none of the three libraries in this project's corpus operates a multi-branch system, which may partly explain this absence relative to what a larger, multi-branch city library system's CBA might contain). All three documents have standard layoff/recall provisions (seniority-based, by classification), with Franklin's explicitly allowing "bumping" into another classification the displaced employee is qualified for and a defined recall window (up to 24 months); no document ties a layoff or staffing-reduction provision to any explicit budget-crisis or service-reduction narrative.

### Peer-community comparability

**No document in this scan contains peer-community, "surrounding," or "comparable communities" wage-comparison language.** This is a third independent replication of the identical finding already documented for every `public_works` and `clerical_admin` document in this project's corpus — comparator language remains, so far, exclusively a feature of this project's two Somerville police JLMC award documents, not of any CBA-only document across four different non-safety and DPW occupation classes now reviewed.

### Grievance vs. interest arbitration

All three documents have ordinary grievance-arbitration procedures (multi-step internal review culminating in binding arbitration for unresolved contract-interpretation disputes), consistent with this project's `interest_arbitration_flag=0` coding for all three rows. **Wayland's grievance procedure is the most fully documented of the three**, explicitly reaching a fifth step ("Arbitration") that routes unresolved grievances to the American Arbitration Association under its Voluntary Labor Arbitration Rules — a genuine, functioning grievance-arbitration mechanism, but expressly scoped to "the interpretation or application of the specific provisions of this Agreement" (per the document's own grievance definition), not to unresolved terms of a new contract. This is correctly *not* interest arbitration, and the existing `interest_arbitration_flag=0` coding for this row is confirmed accurate by this direct reading. Seekonk's and Franklin's grievance procedures are structurally similar (multi-step, culminating in arbitration) though less fully detailed in the extracted text.

### Budget/governance language

**Board of Library Trustees is named explicitly in Seekonk's document** (as one of the bodies through which the Town exercises management rights, per above) — the only one of the three documents to name a library-specific governance body directly. No document contains explicit Municipal Appropriation Requirement, state-aid, or library-specific budget-formula language (consistent with the expectation that this is a municipal/state administrative matter external to a bargaining-unit CBA, not something a union contract would itself restate — see §8 for the external verification of this mechanism).

### Document-by-document summary

- **`ma_seekonk_library_2023`**: shortest and least procedurally detailed of the three; no reclassification, no degree/credential incentive, no volunteer language; the only document naming the Board of Library Trustees explicitly.
- **`ma_franklin_library_2022`**: the richest document in this scan for credentialing (explicit MLS/MLIS stipend tier), composition discipline (explicit page exclusion), and volunteer-substitution protection (Article 28); has a fully documented reclassification process.
- **`ma_wayland_library_2020`** (read via this session's OCR): the richest document for classification granularity (seven library grades plus a custodian grade, ten-step schedule) and the only one with a fully documented, five-step grievance-to-AAA-arbitration procedure; includes a facilities role (custodian) inside a library-specific unit, an unusual composition finding.

## 6. Upward wage-pressure mechanisms

**Professional credentials (MLS/MLIS).** Confirmed as a real, if unevenly applied, pay mechanism in this project's own corpus (Franklin's explicit degree-tier stipend). **Counterargument:** the stipend, where it exists, is a fixed add-on comparable in kind to a teacher's lane placement rather than a separate, higher base-pay track, and Seekonk's document shows this incentive is not universal even among this project's own three cities. **Evidence that would distinguish it:** whether MLS/MLIS-credentialed staff in a given city occupy meaningfully higher pay grades than non-credentialed staff performing similar public-facing duties, versus the credential functioning mainly as a hiring preference with only a modest stipend once employed.

**Specialized public-service roles (children's services, reference, digital services, cataloging, community programming).** Confirmed as real, distinct titles in this project's corpus (Wayland's "Computer Specialist/Reference Librarian," Franklin's "Programming and Community Outreach Librarian," multiple children's/youth-services titles at both cities). **Counterargument:** title specialization alone does not establish a wage premium — Wayland's grade table shows several specialized titles (Bibliographic Services, Reference, Computer Specialist/Reference, Head of Circulation) clustered at the *same* L-4 grade as more general administrative work, suggesting specialization by itself may not move a title to a higher pay tier absent a separate credential or supervisory scope. **Evidence that would distinguish it:** whether specialized titles cluster at higher grades than generalist titles within the same city's own classification schedule (as Wayland's data would allow testing directly) or at the same grade (as this session's reading of Wayland's own appendix suggests, tentatively, for several specialized-but-non-supervisory titles).

**Recruitment/retention.** Not directly evidenced in any of this project's three corpus documents (no vacancy, applicant, or "hard to fill" language found in the extracted text of any of the three). National context (§9) shows real, if easing, staff-loss pressure reported by public libraries generally. **Counterargument:** national survey data on staff losses does not establish that any of this project's three specific cities experienced comparable pressure. **Evidence that would distinguish it:** municipal budget/personnel reports or news coverage specific to these three libraries — not available in this project's current corpus or scope.

**Cost of living.** A general, non-occupation-specific pressure noted here for completeness, consistent with how this mechanism has been treated in every prior non-safety mechanism memo.

**Workload changes from digital services, programming, and expanded community/social-service functions.** Wayland's "Computer Specialist/Reference Librarian" title and Franklin's "Programming and Community Outreach Librarian" title are direct, corpus-confirmed evidence that library job design has already incorporated digital-services and community-programming functions as distinct, named roles, not merely as add-on duties to a traditional reference/circulation job description. **Counterargument:** the mere existence of these titles does not establish that the underlying workload has *grown* relative to some prior baseline (this project's corpus does not include a predecessor contract for any of these three cities that would allow a before/after comparison), nor that any wage response has followed. **Evidence that would distinguish it:** a predecessor-contract comparison (not available for any of these three cities) or direct municipal reporting on service-scope changes over time.

**Classification/reclassification pressure.** Confirmed as a real, corpus-documented mechanism at Franklin and Wayland (§5), structurally identical in design to the DPW and clerical/admin reclassification processes already documented elsewhere in this project's corpus (director-initiated review, administrative-authority approval, new-anniversary-date/step-eligibility consequence). **Counterargument:** as already noted for DPW and clerical/admin, reclassification-driven change is not library-specific — the mechanism recurs identically across multiple occupation classes in this project's corpus, so its presence tells us about a general Massachusetts municipal classification-system feature, not something distinctive to library work. **Evidence that would distinguish it:** frequency/outcome data on library-specific reclassification requests, which (as for DPW and clerical/admin) is internal administrative data not published in CBA text.

**Pay compression.** Not evidenced in any of this project's three corpus documents. National context (§9, the 2006 ALA-APA non-MLS salary study) suggests a long-documented, if dated, compression pattern between credentialed and non-credentialed library staff nationally. **Counterargument:** a 2006 national study cannot be assumed to describe current Massachusetts conditions, and no minimum-wage-driven compression language (of the kind found in the clerical/admin corpus scan's Century Foundation citation) has been identified specifically for library work in this session. **Evidence that would distinguish it:** a current, Massachusetts-specific, or this-project's-cities-specific compression study — not identified this session.

**Public-facing service demands.** Real and structurally embedded in every title in this project's corpus (all are public-service-facing by definition), but this is a description of the job, not by itself a wage-pressure mechanism distinct from the salience question taken up directly in §7.

**Competition with schools, universities, nonprofits, or private-sector information/administrative roles.** Not directly evidenced in this project's corpus (no comparator-employer language found in any of the three documents). This is a plausible national mechanism (library and information-science credential holders can compete for archival, records-management, and information-services roles outside public libraries) but was not independently verified this session beyond general labor-market reasoning.

## 7. Wage-restraint or alternate wage-translation mechanisms

**Lower public salience than police/fire and teachers.** No document in this project's corpus contains political-salience framing (expected, since CBA text rarely contains this kind of language for any occupation class reviewed so far). **Counterargument:** as already cautioned for DPW and clerical/admin, salience should be treated as function-dependent, not uniformly low — a library closure or a controversial collection-development/programming decision can generate real local political attention (e.g., a book-challenge controversy, which is a well-documented national phenomenon in library discourse, though not evidenced in this project's own city-specific sources this session). **Evidence that would distinguish it:** comparative local-news-coverage-volume analysis, not attempted in this project for any occupation class to date.

**Service deferral via reduced hours, branch closures, delayed programming, reduced acquisitions, or queues/backlogs.** **Not directly evidenced in any of this project's three corpus documents** — no hours-reduction, branch-closure, acquisitions-budget, or backlog language was found in the extracted text of any of the three CBAs. This is consistent with the general pattern (also found in the DPW and clerical/admin scans) that service-level and budget-narrative content of this kind lives in budget documents and annual reports, not in the labor contract itself, not evidence that the mechanism is absent in practice. **Counterargument:** the complete absence of this language across all three documents could also simply reflect that a labor contract is the wrong document type to look for it in, regardless of whether the underlying phenomenon occurs. **Evidence that would distinguish it:** municipal library annual reports or budget documents for these three (or any) Massachusetts cities — not reviewed this session.

**Part-time and non-standard staffing as a structural buffering mechanism.** Confirmed, directly and extensively, in all three corpus documents (§5) — part-time employment is a permanent, contractually defined category in every one of this project's three library bargaining units, not an occasional exception. **Counterargument:** the presence of a part-time category does not by itself establish that part-time staffing is *used to buffer* wage pressure rather than simply reflecting the library's inherent staffing model (many public libraries have always used part-time staff for evening/weekend coverage, independent of any wage-restraint motive). **Evidence that would distinguish it:** whether the *share* of part-time staffing in a given library's workforce has grown over successive contract cycles (not testable with this project's current single-cycle-per-city corpus) or whether part-time positions are specifically used to avoid a reclassification or wage increase that a full-time position would otherwise trigger (not evidenced in any of the three documents).

**Pages and volunteers as a lower-cost substitute for covered staff.** **Directly contradicted, not confirmed, by the strongest available corpus evidence.** Franklin's Article 28 explicitly prohibits replacing bargaining-unit members' hours with volunteer help without union agreement — the single clearest textual signal in this entire scan, and it runs in the *opposite* direction from a naive "volunteers buffer library staffing costs" hypothesis. Pages are confirmed excluded from bargaining-unit coverage at Franklin (a composition fact, not a buffering mechanism per se — see §5) but this project's corpus contains no wage or staffing-level data for pages specifically, so no claim can be made about whether page hiring has *grown* as a substitute for covered positions. **Counterargument to the "contradicted" framing itself:** one contractual anti-substitution clause at one of three cities does not establish that volunteer or page substitution never occurs in Massachusetts libraries generally, only that at least one bargaining unit has successfully negotiated against it. **Evidence that would distinguish it:** page/volunteer headcount and hours data over time for any of this project's cities — not available in this project's current corpus or scope.

**Classification and internal-equity systems.** Confirmed as a real, corpus-documented restraint-and-channel mechanism (§5, §6) at Franklin and Wayland, structurally identical to the DPW and clerical/admin classification systems already documented. **Counterargument:** as with the upward-pressure framing of the same mechanism, this is a general Massachusetts municipal classification-system feature, not library-specific.

**Budget constraints, general and library-specific.** Library positions compete with police, fire, teachers, DPW, and clerical/admin for the same Proposition 2½-limited municipal levy capacity already discussed in every prior non-safety mechanism memo. **A library-specific wrinkle, verified externally (§8):** Massachusetts municipal libraries participate in a state certification regime (the Municipal Appropriation Requirement) that penalizes a municipality's library specifically (via loss of state aid eligibility) if its library appropriation falls below roughly 102.5% of the prior three years' average — a real, library-specific budget floor, but a weaker and more waivable one than teachers' Chapter 70/net-school-spending floor, and one that constrains the *library's overall appropriation*, not any specific bargaining unit's wage line within it.

**Library trustees or separate governance structures.** Confirmed present in name only within this project's corpus — Seekonk's management-rights clause names the "Board of Library Trustees" as one of several bodies through which the Town exercises management authority, and every Massachusetts public library certified for state aid is required (per MBLC's own certification standards, §8) to have its Board of Trustees chair sign the annual compliance form. **Counterargument:** naming a Board of Trustees in a management-rights clause does not establish that the Trustees, rather than the ordinary town administration, actually control wage-setting outcomes — in every document reviewed, the Town Administrator (not the library board) is the party actually negotiating and administering the CBA. **Evidence that would distinguish it:** whether any of this project's cities' library trustees have independent budget-setting or override authority distinct from the town's ordinary appropriation process — not established by any source reviewed this session.

**Professionalization without a strong wage premium.** This is the mechanism this memo is best positioned to flag as a hypothesis, not confirm as a finding, given the mixed corpus evidence (Franklin's real but modest stipend; Wayland's education-leave-only treatment; Seekonk's complete absence of degree-linked pay) plus the national context in §9 (BLS's median-wage gap between Librarians, $64,320, and Library Assistants, Clerical, $17.31/hour ≈ roughly $36,000/year full-time-equivalent — a real gap, but one that a dated 2006 ALA-APA study suggests can compress over time between entry-tier credentialed and senior non-credentialed staff). **Counterargument:** a wage gap between two BLS occupational categories nationally does not establish anything about whether *this project's own three cities'* credentialed and non-credentialed library staff show a comparable or a compressed gap — this project's own corpus does not contain enough granular, title-specific wage data (beyond Wayland's grade table, which does not label any single grade as "MLS-required") to test this directly.

**Gendered occupational valuation — explicitly flagged as a hypothesis requiring evidence, not asserted as fact, per the same discipline already applied to clerical/admin.** Librarianship is, like clerical/administrative work, a historically and currently female-dominated occupational category in the United States, and the same comparable-worth/occupational-segregation literature already cited in the clerical/admin mechanism memo (Paula England's research on occupational feminization and wages; IWPR's occupational-segregation research) is, in principle, equally applicable to library work as a professional field. **This memo does not assert that gendered occupational valuation explains any wage pattern in this project's own Massachusetts cities.** No occupational gender-composition data matched to wage data was reviewed or is available for any of this project's three library corpus rows. This is listed here as a hypothesis that would require dedicated, direct evidence (occupational gender-composition data matched to wage data, ideally with a comparable-worth-style skill/complexity comparison against a male-dominated municipal occupation of similar credential requirements) before being treated as an explanatory mechanism, per the task's explicit instruction and the identical discipline already applied to `clerical_admin_gendered_valuation` (folded into CA13 in the living checklist).

**Public support for libraries that may not translate into wage power.** This is a real, if unevidenced-for-this-project's-cities, national pattern worth naming precisely: libraries are consistently among the most trusted and well-regarded local government institutions in general public-opinion research (a claim this memo does not independently re-verify this session, consistent with the general esteem-is-not-supply-or-wage-power caution already applied to firefighters and teachers), while the national compensation data in §9 shows library wages, particularly for entry-tier credentialed librarians, have been reported as flat relative to inflation. **Counterargument:** public esteem and wage power are logically distinct claims, and this project has applied this same caution to every prior occupation group (firefighters, teachers) — restating it for library work is not a new finding, but the alignment of a well-documented national compensation-stagnation claim (§9) with this caution is a genuinely useful, if not project-specific, illustration.

## 8. Massachusetts institutional context

**Library employees appear to fall under the ordinary Chapter 150E route, consistent with (and now extending) the pattern already independently verified three times for teachers, DPW, and clerical/admin.** The Massachusetts Municipal Association's own Select Board Handbook — the same source used to verify clerical/admin's JLMC ineligibility in `ma_clerical_admin_bargaining_impasse_context_2026-07-05.md` — states directly that "the overwhelming majority of municipal employees in Massachusetts are organized under Chapter 150E, and it is common for clerical **and library** employees to be union members," explicitly naming library employees in the same ordinary-bargaining category as clerical staff. No source reviewed this session (the MMA Handbook, Mass.gov's Chapter 150E and JLMC pages, or this project's own three library CBAs) describes any library-specific impasse-resolution body, compulsory arbitration process, or JLMC-equivalent backstop. This project's own corpus independently corroborates this: all three `library` rows' `binding_arbitration_statute` fields already correctly cite `MA G.L. c. 150E` (never the JLMC statute cited by every police/fire row), and every arbitration clause found in the three documents is grievance-scoped (§5), not interest-arbitration-scoped. **This session's verification of the library-specific case is somewhat less exhaustive than the dedicated single-occupation institutional memos already completed for teachers, DPW, and clerical/admin** (each of which is a standalone memo); it is folded into this combined memo instead, consistent with library's practical status as requiring less new verification work than a dedicated institutional memo, given how directly the MMA Handbook's own language extends the identical, already-three-times-verified finding to library employees by name.

**No library-specific JLMC-like backstop was identified.** JLMC eligibility remains confirmed, across every institutional source this project has reviewed to date, as limited to police officers and firefighters specifically — library work has no plausible eligibility argument under any reading of the sources reviewed this or prior sessions.

**Library trustees/governance structures affect the budget channel, not the bargaining-impasse channel, based on the sources reviewed this session.** The Massachusetts Board of Library Commissioners (MBLC) administers a **Municipal Appropriation Requirement (MAR)**: to remain certified for the State Aid to Public Libraries Program (and thereby retain eligibility for state library aid funds), a municipality's library appropriation in a given fiscal year generally may not fall below roughly 102.5% of the average of the preceding three years' library appropriations, subject to waivers the MBLC may grant in certain circumstances; annual compliance is certified by the signature of the library's Board of Trustees chair on a compliance form, alongside other standards (materials-expenditure minimums, librarian certification under 605 CMR 3.00, and minimum standards of free public library service). **This is a real, library-specific, externally-verified budget mechanism, but it is a weaker floor than teachers' Chapter 70/net-school-spending requirement** in at least two respects worth stating precisely: it is expressed relative to the municipality's own recent appropriation history (a rolling local baseline), not an externally-calculated adequacy formula the way Chapter 70's foundation budget is; and non-compliance results in loss of a *discretionary state aid grant program*, not a legal spending mandate enforceable the way net school spending compliance is treated for schools. **This session did not verify** whether any of this project's three library cities (Seekonk, Franklin, Wayland) have ever been out of compliance with the MAR, been granted a waiver, or had this mechanism bind in a way that affected library staff wages specifically — that would require reviewing MBLC compliance records or these cities' own budget history, neither of which was done this session.

**Library workers appear institutionally closer to clerical/admin and DPW than to teachers or police/fire, on the specific dimension of impasse-resolution access — but with their own distinct budget-governance wrinkle teachers, DPW, and clerical/admin do not share.** Library shares the identical "no JLMC access, ordinary Chapter 150E route" finding with clerical/admin and DPW (and, on this specific dimension, with teachers too, per the compulsory-vs-voluntary distinction already established). Where library differs from all four other groups is the MAR — a state-administered, library-specific appropriation floor tied to a rolling local baseline, structurally distinct from Chapter 70's state-formula floor (teachers), the absence of any comparable floor for DPW's general-fund or clerical/admin's general-fund budgeting, and police/fire's JLMC-award-driven funding obligation.

## 9. National context

**Bounded desk research this session, clearly separated from Massachusetts-specific and current-corpus evidence:**

- **Staffing pressure, real but easing.** The Public Library Association's 2024 Public Library Staff Survey found 18% of public libraries reported losing staff positions in the prior 12 months, down from 22.6% in the 2021 survey, with city libraries (29.1%) losing positions at a notably higher rate than suburban (18%) or town/rural (17.1%) libraries ([ALA/PLA, PLA Releases Results from the 2024 Public Library Staff Survey](https://www.ala.org/news/2025/08/pla-releases-results-2024-public-library-staff-survey)). This is national survey data, not evidence about any of this project's three specific cities.
- **Compensation stagnation, specifically flagged.** The same survey/ALA reporting on library labor trends states that beginning librarian salaries "have been virtually flat" relative to inflation, and that library director salaries "have not recovered to pre-Great Recession levels," with a reported 2021 median annual salary of $79,022 for public library directors and $41,864 for beginning librarians ([ALA, Labor Trends & Statistics for Library Workers](https://www.ala.org/educationcareers/careers/stats)).
- **MLS employment pattern.** The same source reports that public libraries employ non-MLS "Associate Librarians" roughly ten times more often than MLS-credentialed "Beginning Librarians" — a striking, quantified illustration of the professionalization/composition point this memo raises in §7 and §6 (most public library staff, nationally, are not MLS-credentialed, even though the credentialed tier anchors the profession's public identity).
- **A dated but directly on-point compression finding.** A 2006 ALA-APA non-MLS salary survey (via the Library Research Service) found the average non-MLS "Associate Librarian" salary ($33,680) was only $6,346 below the average MLS "Beginning Librarian" salary ($40,026) in public libraries at that time, with the report's own framing describing this narrow gap as a lack of adequate recognition for the credentialed tier rather than generous treatment of the non-credentialed tier ([Library Research Service, Non-MLS Salaries in Public Libraries Disparate](https://www.lrs.org/fast-facts-reports/non-mls-salaries-in-public-libraries-disparate/)). **This is a 20-year-old data point and should not be read as describing current conditions**; it is included because it is the most directly on-point, quantified national illustration of the credential-compression mechanism this memo hypothesizes, not because it is current evidence.
- **BLS occupational wage context (source context only, not processed into a panel).** BLS's May 2024 Occupational Employment and Wage Statistics reports a median annual wage of $64,320 for "Librarians and Library Media Specialists" nationally (employment projected to grow a modest 2% from 2024-2034) and a median hourly wage of $17.31 for "Library Assistants, Clerical" (SOC 43-4121; the broader "Library Technicians and Assistants" category is projected to *decline* 7% in employment from 2024-2034, with roughly 25,800 annual openings driven by turnover, not growth) ([BLS OOH, Librarians and Library Media Specialists](https://www.bls.gov/ooh/education-training-and-library/librarians.htm); [BLS OOH, Library Technicians and Assistants](https://www.bls.gov/ooh/education-training-and-library/library-technicians-and-assistants.htm); [BLS OEWS, 43-4121 Library Assistants, Clerical](https://www.bls.gov/oes/2023/may/oes434121.htm)). BLS's own Occupational Outlook Handbook states a master's degree in library science is the standard credential for professional librarian positions. These are national occupational categories, not bargaining-unit data, and are cited here as source context only, consistent with this project's standing rule against building an OEWS panel in a mechanism-development session.
- **Service deferral (hours/branch/program reductions).** No specific national source on library hours reductions, branch closures, or program cuts was pulled this session beyond the staff-loss survey data above; this remains an area flagged for future desk research, not a closed finding.
- **Gendered occupational valuation.** No new national source specific to library work was pulled this session beyond the general comparable-worth/occupational-segregation literature already cited for clerical/admin (Paula England, IWPR); this hypothesis is restated, not newly evidenced, for library work specifically (§7).

**What this session explicitly did not verify:** any Massachusetts-specific library staffing, vacancy, or compensation survey (the MBLC's own data holdings were not reviewed for this beyond the certification/MAR pages already cited); whether any of this project's three cities' libraries have experienced a reported staff loss, hours reduction, or branch-service change; and any book-challenge or collection-development political-salience episode for any of this project's cities.
