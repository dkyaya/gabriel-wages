# Clerical / Administrative Source Gap List

**Date:** 2026-07-05
**Scope:** desk-research planning only. Lists the highest-value evidence gaps surfaced by `non_safety_clerical_admin_wage_mechanism_refinement_2026-07-05.md`. Several items below already have a partial public source identified during this session's bounded web search or light corpus read; those are noted. This is a punch list for future sessions, not a completed data build.

For each gap: why it matters, likely source families, a feasibility tag (`desk_research_feasible`, `data_build_feasible`, `likely_hard`), and which dimension it speaks to (wage levels, wage growth, staffing supply, classification, reclassification, compression, automation, workload, service backlogs, or budget constraints).

## 1. Clerical/admin applicant counts over time

**Why it matters:** needed to distinguish an applicant-volume problem from an applicant-qualification problem for clerical/admin roles, mirroring the same distinction already used for police, teacher, and DPW recruitment.

**Likely source families:** individual Massachusetts municipal HR/civil-service application records; MissionSquare Research Institute or similar national local-government workforce surveys (national context only, not Massachusetts-specific).

**Feasibility:** `likely_hard` at a systematic level — no centralized national or Massachusetts-specific clerical/admin applicant-tracking survey was identified this session, unlike NCES's teacher-vacancy snapshot; `data_build_feasible` only if narrowed to a small number of this project's specific cities and their own HR records.

**Dimension:** staffing supply.

## 2. Clerical/admin vacancy rates

**Why it matters:** the most direct staffing-strain measure, and the clerical/admin analog to the police, teacher, and DPW vacancy data already used elsewhere in the project's mechanism map.

**Likely source families:** municipal budget books (personnel/vacancy line items); municipal annual reports; MissionSquare's cross-occupation "hard to fill" reporting (national context, partially identified this session, not clerical/admin-specific and not Massachusetts-specific).

**Feasibility:** `data_build_feasible` for this project's specific cities if their annual town reports or budget books include clerical/admin staffing/vacancy detail; `likely_hard` for a systematic multi-city Massachusetts series.

**Dimension:** staffing supply.

## 3. Reclassification requests or studies

**Why it matters:** directly tests the `clerical_admin_reclassification_pressure` hypothesis — whether reclassification requests for clerical/admin titles are common, how they are resolved, and how their outcomes compare to across-the-board wage increases in dollar terms.

**Likely source families:** this project's own already-collected `clerical_admin` CBAs (Boston's contract already documents the formal OHR Classification and Compensation Unit review process, per this session's light read); municipal HR/personnel board records; classification-and-compensation study reports if municipalities commission them periodically.

**Feasibility:** `data_build_feasible` for confirming the process described in Boston's contract in more detail; `likely_hard` for obtaining actual reclassification-request outcome data (approval rates, dollar impact), since this is typically internal HR administrative data, not published.

**Dimension:** classification / reclassification.

## 4. Pay-grade and step-schedule structures

**Why it matters:** needed to confirm whether clerical/admin classification systems are structured similarly to the DPW classification systems already documented in this project's corpus (grade-and-step, with promotional/reclassification movement) and whether clerical/admin schedules span multiple departments under one unified system, as this memo hypothesizes based on the Boston example.

**Likely source families:** this project's own already-collected `clerical_admin` CBAs and published salary schedules; municipal classification-and-compensation plan documents where publicly posted.

**Feasibility:** `data_build_feasible` — this project already holds three `clerical_admin` CBAs; a fuller document-by-document review (mirroring the dedicated DPW corpus-scan session) would likely close most of this gap without new source acquisition.

**Dimension:** classification.

## 5. Pay compression evidence

**Why it matters:** tests whether the gap between entry-level and senior clerical steps has actually narrowed over time in specific Massachusetts municipalities, as opposed to the general national/multi-state cost estimate already identified this session.

**Likely source families:** municipal salary schedules over multiple contract cycles (this project's own corpus could support this for cities with multiple `clerical_admin` contract cycles, though currently only Worcester, Boston, and Seekonk each have one cycle collected); The Century Foundation's minimum-wage-compression cost report (already identified this session, national/multi-state including Massachusetts, not city-specific).

**Feasibility:** `likely_hard` with the current corpus (only one contract cycle per city collected so far, insufficient for a within-city compression trend); `desk_research_feasible` for the general multi-state estimate already closed this session.

**Dimension:** compression / wage levels.

## 6. Minimum-wage compression evidence, Massachusetts-specific

**Why it matters:** Massachusetts is explicitly one of the jurisdictions covered by The Century Foundation's minimum-wage-compression cost estimate (~0.6% weighted average annual cost across six jurisdictions), but this session did not extract the Massachusetts-specific figure from that estimate, only the multi-state weighted average and the range (0.2% to 1.0%).

**Likely source families:** The Century Foundation's full report (already identified this session at the summary level; a direct read of the full report would be needed for the Massachusetts-specific figure); Massachusetts minimum-wage statutory schedule and effective dates.

**Feasibility:** `desk_research_feasible` — the source is already identified and publicly available; only a more detailed read is needed to extract the Massachusetts-specific figure.

**Dimension:** compression / budget constraints.

## 7. Turnover/retention evidence

**Why it matters:** needed to confirm whether the national local-government turnover pattern identified this session (MissionSquare/BLS JOLTS data showing rising public-sector quits, with non-competitive compensation cited as the top exit reason) applies specifically to clerical/admin roles and to Massachusetts municipalities, as opposed to the cross-occupation, national pattern already found.

**Likely source families:** MissionSquare Research Institute (already identified this session, national and cross-occupation, not clerical/admin-specific or Massachusetts-specific); Massachusetts municipal HR turnover reports if published.

**Feasibility:** `desk_research_feasible` for further MissionSquare or BLS JOLTS detail at the national level; `likely_hard` for Massachusetts-specific, clerical/admin-specific turnover data, since this is not typically published at that granularity.

**Dimension:** staffing supply.

## 8. Workload/backlog evidence

**Why it matters:** directly tests the `clerical_admin_service_backlog_absorption` hypothesis — whether clerical/admin staffing shortfalls actually show up as measurable service delays (permit turnaround times, records-request response times, reduced office hours) in specific Massachusetts municipalities.

**Likely source families:** municipal annual reports and department performance reports where published; local news coverage of specific service delays (illustrative, not systematic proof); public-records-request response-time data if tracked and published.

**Feasibility:** `likely_hard` — no centralized source for this kind of service-level data was identified this session; would require city-by-city review of annual reports or direct outreach to municipal departments.

**Dimension:** workload / service backlogs.

## 9. Reduced office hours or service delays

**Why it matters:** a more specific, potentially easier-to-find version of gap 8 — municipalities sometimes publicly announce reduced office hours (e.g., town clerk or treasurer's office hours), which would be a directly observable, dated signal of staffing strain being absorbed through reduced service rather than wage growth.

**Likely source families:** municipal website announcements and local news coverage (illustrative, not systematic proof); municipal annual reports.

**Feasibility:** `desk_research_feasible` for illustrative individual examples if a specific Massachusetts city is targeted; `likely_hard` for a systematic multi-city series.

**Dimension:** workload / service backlogs.

## 10. Automation/digitization/reorganization evidence, Massachusetts-specific

**Why it matters:** this session found general national vendor material describing municipal permitting/records-digitization trends, but no Massachusetts-specific confirmation of which municipalities have adopted these systems or what effect, if any, this has had on clerical staffing levels or workload composition.

**Likely source families:** municipal-software vendor case studies (already identified this session as general/national, treat as marketing material not neutral evidence); individual Massachusetts municipal announcements of new permitting/records systems; Massachusetts Municipal Association materials on municipal technology adoption.

**Feasibility:** `desk_research_feasible` for illustrative individual examples; `likely_hard` for a systematic assessment of automation's effect on clerical staffing or wages.

**Dimension:** automation / workload.

## 11. Shared-service or outsourcing evidence, Massachusetts-specific

**Why it matters:** this session identified Massachusetts's Efficiency and Regionalization grant program and specific examples (shared Regional Sustainability Coordinator across Westford/Carlisle; shared Town Administrator across Berkshire Regional Planning Council/Savoy), but did not confirm whether any of this project's specific cities participate in shared clerical/administrative arrangements.

**Likely source families:** Mass.gov Efficiency and Regionalization grant recipient lists (already identified this session at a general level); individual Massachusetts municipal intermunicipal-agreement records.

**Feasibility:** `desk_research_feasible` — the Mass.gov grant-recipient list is public and could be cross-checked against this project's specific cities directly.

**Dimension:** automation / workload / budget constraints.

## 12. Gendered occupational valuation evidence — treated carefully

**Why it matters:** the mechanism memo explicitly flags gendered occupational valuation ("comparable worth") as a hypothesis requiring dedicated evidence, not an established fact about this project's cities. Closing this gap responsibly would require occupational gender-composition data matched to wage data, and ideally a skill/complexity comparison in the spirit of the comparable-worth literature, rather than an assumption based on general labor-market patterns.

**Likely source families:** academic labor economics/sociology literature on comparable worth and occupational segregation (general background, not Massachusetts-specific or this-project-specific); if available, gender-composition data for specific Massachusetts municipal clerical/admin bargaining units, compared against police/fire/DPW gender composition in the same cities.

**Feasibility:** `likely_hard` for this project's specific cities, since bargaining-unit gender-composition data is not part of this project's current corpus or schema and may not be readily available; `desk_research_feasible` only for the general academic literature background, which should not be treated as a substitute for city-specific evidence.

## 13. Peer-community comparisons

**Why it matters:** tests whether clerical/admin bargaining invokes "surrounding" or "comparable" community wage comparisons the way teacher and (per this project's DPW corpus scan) apparently not DPW bargaining does, and whether any such comparator language is backed by an institutional enforcement mechanism or functions as persuasive rhetoric only, mirroring the question already resolved for teachers and DPW via the JLMC-vs-Chapter-150E-Section-9 distinction.

**Likely source families:** this project's own already-collected `clerical_admin` CBAs (not yet reviewed for this specific language, since this session's read was light and illustrative); municipal HR salary-survey/comparison materials if published.

**Feasibility:** `data_build_feasible` if this project's already-collected `clerical_admin` CBAs are reviewed in full for comparator language, mirroring the dedicated DPW corpus-scan session's approach.

**Dimension:** wage levels / classification.

## 14. Massachusetts examples vs. national context

**Why it matters:** this session relied on a mix of light corpus review (three cities) and national/multi-state web research (MissionSquare, The Century Foundation, municipal-software vendors); future work should be explicit about which claims are Massachusetts-verified versus nationally-plausible-but-unverified for Massachusetts specifically, consistent with the discipline already applied in the teacher and DPW memos.

**Likely source families:** individual Massachusetts city/town clerical/admin classification plans, budget books, and CBAs (illustrative, not comprehensive, per this session's scope instruction); Massachusetts Municipal Association materials.

**Feasibility:** `data_build_feasible` for a small number of illustrative Massachusetts examples via a fuller review of this project's already-collected corpus; explicitly not `desk_research_feasible` at a comprehensive statewide level, consistent with this session's scope boundary against comprehensive statewide coverage.

## 15. Massachusetts clerical/admin impasse-process verification

**Why it matters:** highly likely, by the same institutional logic already established twice (for teachers and for DPW), that clerical/admin bargaining follows the ordinary Chapter 150E Section 9 route with no JLMC-equivalent backstop, since clerical/admin has no plausible JLMC eligibility argument at all (JLMC is confirmed police/fire-specific). However, this was not directly re-verified for clerical/admin specifically this session, and this project's discipline has been to verify each occupation class independently rather than assume institutional findings transfer automatically.

**Likely source families:** Mass.gov DLR/JLMC pages (already used twice this project for teachers and DPW); this project's own `clerical_admin` CBAs' `binding_arbitration_statute` and arbitration-clause fields, which could be checked directly (Seekonk's clerical row already shows `MA G.L. c. 150E` in this field, consistent with the expected finding, though this was not the focus of a dedicated verification session for clerical/admin the way it was for teachers and DPW).

**Feasibility:** `desk_research_feasible` — this is a quick, targeted verification given the institutional pattern already established twice; a short follow-up session could close this without extensive new research.
